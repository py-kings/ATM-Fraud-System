# 💳 # 💳 ATM Transaction Management & Fraud Detection System

A modern Flask + MySQL based ATM Transaction Management & Fraud Detection System with real-time analytics dashboard.

---

## 🚀 Features

- Secure ATM Transactions
- Deposit & Withdraw
- Money Transfer
- Fraud Detection
- Transaction History
- Admin Dashboard
- Dark Mode
- Charts & Analytics

---

## 🛠️ Tech Stack

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js

---
